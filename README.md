# spring-data-gremlin with DSE

This demo shows Apahce Tinkerpop graph application using [spring-data-gremlin](https://github.com/Microsoft/spring-data-gremlin) and DataStax Enterprise.

## Running application

```
mvn spring-boot:run
```
